# 结构体补充
```rs
//main.rs
fn main()
{
    let origin = Point3d::default();
    println!("origin: {},{},{}",origin.x,origin.y,origin.z);
    let point = Point3d{x:99,..origin};//一个包含..的结构体可以用来从其他结构体拷贝一些值或者在解构时忽略一些字段
    println!("point: {},{},{}",point.x,point.y,point.z);
    let Point3d{x:x0,y:y0,..} = point;
    println!("point解构的: x0:{},y0:{}",x0,y0);//x0,y0具名解构

    //字段可变性，使用Cell
    let mut point = Point{x:5,y:Cell::new(6)};
    println!("point.y: {:?}",point.y);
    point.y.set(7);
    println!("point.y: {:?}",point.y);//现在没有提及所有权
}
#[derive(Default)]
struct Point3d
{
    x:i32,
    y:i32,
    z:i32,
}
use std::cell::Cell;
struct Point
{
    // mut x:i32,不支持字段可变性,这是因为：可变性是绑定的一个属性，而不是结构体本身，可以用Cell<T>来模拟可变
    x:i32,
    y:Cell<i32>,
}
```
# 总结
- Cell的用法
- 字段赋值，解构符号: ..
- 使用#[derive(Default)]注解，和default声明结构体
