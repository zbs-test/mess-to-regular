# c++注意点
- 虚析构函数是为了以后`其子类正确调用其析构函数`，`不设置会子类调用析构函数会出错`
- 继承大部分以`public继承`
- 父类指针将成为常客，因为我们`需要父类指针指向子类对象`，从而实现`多态`
- `new出来的是堆对象`，不是栈对象
- `虚函数(虚方法)是用来实现多态的`，`为各类型对象提供本土(类型)方法`
